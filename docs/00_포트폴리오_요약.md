# 포트폴리오 1페이지 요약본 (제출용)

## 프로젝트명: X-ray 의료영상 전처리 시각화 도구

### 개요
OpenCV 기반 **의료영상(DICOM, X-ray) 전처리 알고리즘 비교 도구** 구축
- DICOM Window Level 표준 시각화 지원
- CLAHE/Canny Edge 실시간 파라미터 튜닝
- Before/After 비교 UI + REST API 제공

### 기술스택
`Python` `Streamlit` `FastAPI` `OpenCV` `pydicom` `Pillow`

### 아키텍처
```
[파일 업로드] → [preprocess_core.py] → [Streamlit UI / FastAPI]
  (DICOM/PNG/JPG)    (이미지 처리 로직)     (Before/After 비교)
```

### 주요 성과
| 항목 | 내용 |
|------|------|
| 지원 포맷 | DICOM(.dcm), PNG, JPG, BMP |
| 전처리 | CLAHE 대비 향상, Canny 에지 검출 |
| 시각화 | Window Level (의료 표준) / Min-Max 정규화 |
| API | FastAPI REST API (Base64 PNG 반환) |

### 핵심 기능
- **DICOM 처리**: RescaleSlope/Intercept 자동 적용, 메타데이터 추출
- **실시간 튜닝**: Clip Limit, Tile Grid Size, Canny Threshold 슬라이더
- **이중 인터페이스**: 웹 UI (Streamlit) + REST API (FastAPI)

### 의료영상 도메인 지식 활용
- 방사선사 면허 보유 → DICOM 표준, Window Level 개념 이해
- 의료영상 학사 → X-ray 영상 특성, 전처리 필요성 파악
